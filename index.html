GENFE301 · Fundamentals of Electricity

Electricity

Core concepts, circuits, installation and real troubleshooting.

Core Concepts
Voltage Push of electricity
Current Flow of electricity
Resistance Opposition to flow
Ohm's Law: V = I × R
If device fails: No voltage = no power | No current = break | High current = short
Circuit Types
TypeKey IdeaUse
SeriesSame currentSimple circuits
ParallelSame voltageHome wiring
Parallel is used in houses so one device failing doesn’t stop others
AC vs DC
AC Alternating (homes)
DC Direct (electronics)
Power supply converts AC → DC for computers
Installation & Testing
  • Continuity test → check wire break
  • Voltage test → check supply
  • Current test → check flow
⚡ Electricity Cheatsheet
V = I × R Series: same current Parallel: same voltage AC = alternating DC = direct Test: voltage, current, continuity
GENBE302 · Basic Electronics

Electronics

Components, circuits, testing and troubleshooting.

Passive Components
Resistor Limits current
Capacitor Stores charge
Inductor Magnetic storage
Burnt resistor = open circuit | Bulging capacitor = faulty
Active Components
Diode One-way current
Transistor Switch/amplify
IC Complete circuits
If IC fails → whole system fails
Workshop Tools
  • Multimeter (MOST IMPORTANT)
  • Oscilloscope
  • Soldering iron
Troubleshooting Steps
1. Check power 2. Check current 3. Identify faulty component 4. Test with multimeter 5. Replace
🔌 Electronics Cheatsheet
Resistor = limits current Capacitor = stores charge Diode = one-way Transistor = switch IC = full circuit Always test with multimeter
